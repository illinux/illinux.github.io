## 代码块

```javascript
var modularpattern = (function() {
    // 模块代码
    var sum = 0 ;

    return {
        add:function() {
            sum = sum + 1;
            return sum;
        },
        reset:function() {
            return sum = 0;    
        }  
    }   
}());
alert(modularpattern.add());    // 创建 1
alert(modularpattern.add());    // 创建 2
alert(modularpattern.reset());  // 创建 0
```

## 水平线

HTML `<hr>`元素用于在段落级元素之间创建“突破主题” . 在 markdown中, 可以在`<hr>`里创建以下元素:

* `___`: 三次连续强调
* `---`: 三次连续破折号
* `***`: 三次连续星号
